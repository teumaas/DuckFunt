Before install extension: 
1. Installeren DirectX
2. Installeren XNA Framework 4.0 Redistribution
3. Installeren XNA Game Studio 4.0 Platform Tools
4. Installeren XNA Game Studio 4.0 Shared
5. Installeren XNA Game Studio 4.0.vsix
6. Lanceren the XNA_test solution om te XNA testen

website : http://mxa.codeplex.com/releases/view/117230

- Visual Studio 2013 ultimate is nodig om dit framework te installeren.

- Als je moet kiezen tussen 'repair' en 'remove', kies 'repair'.

- Als VisualStudio niet reageert na het starten van de Test solution, installeer Xna opnieuw en start je je pc opnieuw op.
